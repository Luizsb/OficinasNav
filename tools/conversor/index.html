help Como formatar o Word (.docx)

O template NAVE usa marcadores entre colchetes, não estilos de Título do Word. O conversor reconhece automaticamente:

  • [Título da oficina] — nome e metadados
  • [Texto] — parágrafos da visão geral
  • [x] 7º ano EFAF — ano, duração e eixo (checkboxes marcados)
  • [Quadro-resumo] + tabela — cards Preparar / Criar / Refletir
  • [Etapa 1], [Etapa 2], [Etapa 3]
  • [Seção 1] Apresentação, [Perguntas reflexivas], [Elemento-chave], [Seção 3] Embarque
  • [Título] Chapéu Virtual + etapas 1. …, 2. … (seção Criar)
  • <inserir gif_oficina_4_1> — referência de imagem; figuras embutidas no Word também são extraídas
